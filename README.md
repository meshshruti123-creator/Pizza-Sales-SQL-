# Pizza-Sales-SQL-
This project is a complete SQL-based data analysis of a pizza restaurant's sales data using MySQL Workbench. It involves designing a relational database with 4 tables — orders, order_details, pizzas, and pizza_types — and writing queries ranging from basic to advanced level. The analysis covers total revenue, most ordered pizzas, peak order hours, category-wise sales, cumulative revenue trends, and percentage contributions. This project demonstrates strong SQL skills including JOINs, GROUP BY, aggregate functions, subqueries, and window functions.
